# Workflow — выполнение запроса

```mermaid
flowchart TD
    start(["Запрос пользователя"])

    session{"Сессия существует?"}
    ratelimit{"Лимит запросов не превышен?"}
    domain{"Домен разрешён?"}
    followup{"Ответ на уточняющий вопрос?"}
    catalog{"Вопрос по теме магазина?"}
    intent{"Тип запроса?"}
    clarify{"Нужен уточняющий вопрос?"}

    err1(["Сессия не найдена"])
    err2(["Слишком много запросов"])
    err3(["Домен не разрешён"])
    offtopic(["Вопрос не по теме магазина"])
    clarify_q(["Уточняющий вопрос"])
    result(["Ответ + карточки товаров"])

    c1["Извлечь упомянутые товары (LLM)"]
    c2["Сгенерировать экспертный ответ (LLM)"]
    c3["Поиск товаров по каталогу"]
    c4["Проверить релевантность (LLM)"]

    t2["Определить список нужных товаров (LLM)"]
    t3["Поиск товаров по каталогу"]
    t4["Проверить релевантность (LLM)"]

    start --> session
    session -->|"нет"| err1
    session -->|"да"| ratelimit
    ratelimit -->|"превышен"| err2
    ratelimit -->|"ок"| domain
    domain -->|"нет"| err3
    domain -->|"да"| followup
    followup -->|"нет"| catalog
    catalog -->|"нет"| offtopic
    catalog -->|"да"| intent
    intent -->|"консультация"| c1
    intent -->|"задача"| clarify
    followup -->|"да — пропустить уточнение"| t2
    c1 --> c2 --> c3 --> c4 --> result
    clarify -->|"да"| clarify_q
    clarify -->|"нет"| t2
    t2 --> t3 --> t4 --> result
```

## Количество LLM-вызовов на запрос

| Сценарий | Вызовов | Примерное время |
|---------|:-------:|:--------------:|
| Консультация | 4–5 | 6–12 сек |
| Задача с уточнением | 3 | 5–8 сек |
| Задача без уточнения | 4–5 | 7–14 сек |
| Ответ на уточняющий вопрос | 2–3 | 4–8 сек |
| Вопрос не по теме | 1 | 1–2 сек |

## Поведение при ошибках LLM

| Шаг | При ошибке |
|-----|-----------|
| Проверка тематики | Считаем «не по теме» |
| Классификация типа | Считаем «задача» |
| Уточняющий вопрос | Пропускаем уточнение |
| Список товаров | Пустой список — сообщение об ошибке |
| Фильтр релевантности | Все товары считаются релевантными |
