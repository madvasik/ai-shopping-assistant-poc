# Контекстная диаграмма — AI Shopping Assistant

```mermaid
flowchart TD
    buyer["Покупатель"]
    admin["Админ"]
    shop["Сайт магазина"]
    system["AI Shopping Assistant"]
    openai["OpenAI API"]
    catalog[("Каталог товаров")]
    dashboard["Панель админа"]

    buyer -->|"открывает сайт"| shop
    shop -->|"встраивает виджет"| system
    buyer -->|"задаёт вопросы в чате"| system
    system -->|"LLM-запросы"| openai
    system -->|"поиск товаров"| catalog
    system -->|"пишет логи"| dashboard
    admin -->|"просматривает логи"| dashboard
```

## Границы системы

| Элемент | Входит в систему | Примечание |
|---------|:---:|------------|
| Чат-виджет | ✅ | Встраивается через `<script>` |
| API бэкенд | ✅ | Оркестрация, сессии |
| Поиск по каталогу | ✅ | Работает внутри процесса |
| OpenAI API | ❌ | Внешний сервис |
| Каталог товаров | ✅ | Файл в контейнере |
| Корзина / Заказы | ❌ | Бот только рекомендует товары |
| Авторизация пользователей | ❌ | Не требуется |
